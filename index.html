Workshop Git deel 1

7 september 2022 [Recording]

Git is een belangrijk onderdeel van Software Development. Tijdens deze interactieve workshop leren we je comitten, pushen en repositories aanmaken.

Git aan nieuw Webstorm project toevoegen

Open Webstorm en start een nieuw project. Initialiseer Git in het nieuwe project via onderstaande stappen.

  1. Open terminal in Webstorm.
  2. Voer het git init command in.
  3. Maak in Webstorm project een nieuw bestand aan.
  4. Geef het bestand .gitignore als bestandsnaam.
  5. Zet in dit bestand de verwijzing naar de map: /.idea

Voeg in het project nu een HTML- en CSS-bestand toe.

Commit aanmaken

Geef inhoud aan het nieuwe HTML- en CSS-bestand.

  1. Voer in de terminal het command git status in.
  2. De gewijzigde bestanden worden in rode tekst weergegeven.
  3. Stage deze of enkele bestanden.
  4. Voer voor alle bestanden stagen het command git add . in.
  5. Voer voor een enkel bestand het command git add bestandsnaam in.
  6. Controleer met het command git statusof de bestanden naar groene tekst zijn gewijzigd.
  7. Commit de bestanden met het command git commit -m "omschrijving van project status"

Controleer uitgevoerde Commit

Om te controleren of de uitgevoerde commit succesvol is kan het command git login de terminal ingevoerd worden.

Het geeft een overzicht van alle commits. Toets herhaaldelijk enterom tot het einde van het log te komen. Sluit eventueel af met het command :q
